OAuth callback page for "Obtain private app publishing rights"

This page is the callback URL for the OAuth process initiated by using the Fastlane Plugin for Manage Google Play action get_managed_play_store_publishing_rights. The general process is described in "Get started with custom app publishing > Obtain private app publishing rights"

Result

  • Developer Account ID: (unknown)
  • Developer Account Link: (unknown)

The Developer Account ID is required for further usage as a parameter developer_account_id of the plugin action create_app_on_managed_play_store.

Note that the "Developer Account ID" currently can simply be obtained from the URL you end up on when logging in to the Google Play Console as well - which is returned as "Developer Account Link" here. If you ever forget or loose it, just look there.

Attention: The data is only output for you, not captured or logged in any way. The sourcecode for this plain HTML page is at https://github.com/janpio/fastlane-plugin-managed_google_play/blob/gh-pages/index.html.
